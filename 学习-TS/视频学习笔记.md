## TS介绍

- js的超集。遵循ES6、ES5规范。
- 适用大型项目
- Nodejs框架Nestjs、midway中用的是TS语法

## TS安装

```typescript
npm install -g typescript
```

## VS自动编译.ts文件

- 创建tsconfig.json 文件 tsc --init 生成配置文件
- 改json文件输出路径
- 终端ts监视
